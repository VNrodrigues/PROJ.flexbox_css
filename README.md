## Projeto utilizando Flexbox em CSS

Neste projeto, aprendi a posicionar elementos na página web utilizando o flexbox em CSS.

Conteúdos estudados:

- Fundamentos do flex box;
- Display:flex;
- Flex direction;
- Flex wrap;
- Flex flow;
- Flex grow;
- Flex basis;
- Flex shrink;
- Justify content;
- Align items;
- Align content;
- Align self;
